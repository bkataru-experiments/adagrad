# adagrad

autograd in ada, by hand, as instructed and guided by deepseek-v4-pro

> a complete, from-scratch reverse-mode automatic differentiation ("autograd") engine written in Ada, together with optimizers, layer abstractions, and a small training demo

## getting started

install gnat either by itself or via alire, then

```shell
$ gnatmake -P grad.gpr
```
